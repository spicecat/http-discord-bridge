# HTTP Discord Bridge

**HTTP Discord Bridge** is used to enable http requests to control a Discord account. 

# **Features**: 
- Discord bot and self-bot support
- Free remote server with [AWS Lambda] (https://aws.amazon.com/lambda/)

## Disclaimer 
 Automating Discord User accounts (self-bots) violates Discord Terms of Service & Community guidelines. Use at your own risk. Read about Discord [Terms Of service](https://discord.com/terms) and [Community Guidelines](https://discord.com/guidelines)

## Usage
1) Replace the value of `TOKEN` in `/var.js` with your token. 
2) Run `npm run start`
